# NTU-Course-Helper 台大選課小幫手
從台大課程網爬課程資料，根據輸入空堂表及系所代號輸出可以選的通識、國文、外文、體育、系所課程。

# 環境設定
## 執行檔
於Release中下載

## Python環境
### Library install
批量安裝：
> pip install -r py-require.txt

單個安裝：
> pip install pandas
> 
> pip install beautifulsoup4
> 
> pip install requests
> 
> pip install styleframe
